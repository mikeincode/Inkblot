# MVP Specification

## Scope

Minimal Viable Product focuses on core universe creation and collaboration.

## Core Features

1. **Universe Management**
   - Create and manage universes
   - Define basic metadata (name, description, genre)

2. **Branching Timelines**
   - Create timeline branches
   - View branch history
   - Compare branches

3. **Factions**
   - Create and join factions
   - Simple faction directory

4. **Canon Review**
   - Submit narrative changes
   - Community voting on canon decisions
   - Approval workflow

5. **User Profiles**
   - Basic profile information
   - Contribution history
   - Faction membership

## Out of Scope (v2+)

- Advanced permission systems
- Real-time collaboration
- Full API marketplace
- Analytics dashboard
