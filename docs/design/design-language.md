# Design Language

## Visual Identity

Inkblot uses a mystical, mythology-inspired design language.

## Color Palette

Referenced from attached assets:
- Deep purples and blacks (Ashfall Kingdoms)
- Neon accents (Neon Hollow)
- Ocean blues (Hollow Sea)
- Gold/crimson (Eclipse Protocol)

## Typography

- Headers: Bold, commanding presence
- Body: Clean, readable sans-serif
- Code: Monospace for technical content

## UI Patterns

- Card-based layouts for content
- Tree/node visualization for branches
- Modal dialogs for decisions
- Timeline visualization for epochs

See component library in `apps/web/src/components/ui/`
