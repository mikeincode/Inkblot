# Inkblot Terminology

## Narrative Concepts

| Term | Definition |
|------|------------|
| Universe | A complete fictional world/setting |
| Canon | Officially accepted lore |
| Branch | A timeline fork or divergence point |
| Shadow Narrative | Alternative/rejected story |
| Epoch | Major time period within a universe |

## Community Concepts

| Term | Definition |
|------|------------|
| Faction | Organized group within a universe |
| Chamber | Discussion/decision-making space |
| Contributor | Creator who adds to a universe |
| Curator | User with editorial responsibilities |
