# Inkblot

> Where Worlds Take Root.

Inkblot is a collaborative mythology platform where stories evolve as living ecosystems.

Instead of isolated posts or static fiction, worlds grow through:
- Trunks
- Branches
- Timelines
- Nodes
- Factions
- Canon
- Shadow evolution

Inkblot combines collaborative storytelling, worldbuilding, governance systems, alternate timelines, lore discussions, and community mythology into a single living structure.

---

# Core Philosophy

Every world begins with a single root.

Inkblot is built around the idea that stories are not static objects — they are living systems shaped by:
- creators
- contributors
- interpretation
- conflict
- mutation
- collective memory

The platform treats mythology as something that can evolve organically through community participation.

---

# The Tree & The Shadow

## The Tree
The governed mythology structure.

The Tree contains:
- Canon lore
- Approved Nodes
- Official timelines
- Curated Branches
- Keeper governance

The Tree represents structured continuity and intentional worldbuilding.

---

## The Shadow
The ungoverned mythology ecosystem.

The Shadow contains:
- Alternate timelines
- Mutations
- Divergent interpretations
- Experimental lore
- Community-driven evolution

The Shadow has no canon.
Only consensus.

---

# Core Terminology

| Term | Meaning |
|---|---|
| Universe | A complete mythology ecosystem |
| Trunk | The primary canon continuity |
| Branch | A divergence from the canon path |
| Timeline | Chronological narrative progression |
| Node | A single contribution or lore entry |
| Keeper | A universe maintainer/moderator |
| Chamber | Localized discussion space |
| Faction | Interpretive ideological group |
| Canonize | Approve into official continuity |
| Mutation | A transformed or remixed Node |
| Shadow | The unofficial evolutionary ecosystem |

# Current Systems

## Universe Creation
Creators establish:
- world identity
- governance philosophy
- ecosystem rules
- creation myths

---

## Tree / Shadow Ecology
Worlds evolve through:
- canon structure
- branching timelines
- shadow mutations
- faction interpretation

---

## Node Contributions
Contributors can:
- submit lore
- create mutations
- grow branches
- propose timelines
- expand mythology collaboratively

---

## Canonization Review
Keepers review submissions and determine whether a Node:
- enters canon
- becomes a Branch
- returns to Shadow
- is archived

---

## Chambers
Localized discussion spaces tied directly to:
Universe → Branch → Timeline

Discussions include:
- Canon Debates
- Lore Theory
- Mutation Proposals
- Timeline Analysis
- Shadow Rumors

---

## Factions
Interpretive ideological groups that shape mythology through:
- philosophy
- alignment
- influence
- resonance
- collective interpretation
